# Submitting Your Task

There will be an **Assignment** on **Google Classroom** for you to submit your final project files to. You will need to do the following:

{% stepper %}
{% step %}
### Make Your Repository Public

Make sure your **GitHub repository** is **PUBLIC** so that **YOUR TEACHER CAN VIEW IT!**

{% embed url="https://drive.google.com/file/d/1HZrvCnkyhVgbfT_n7qFZfrNyga5dAVJL/view?usp=sharing" %}
Making Your GitHub Repository Public
{% endembed %}
{% endstep %}

{% step %}
### Check All Files

Check all of your files on your repository **BEFORE SUBMISSION.**&#x20;

Make sure all files are there, clone the whole repository to your PC and **TEST THOROUGHLY.**
{% endstep %}

{% step %}
### Submit to Google Classroom

Submit your **PUBLIC Github Repository** link to **Google Classroom BY THE DUE DATE**.

**If it is late, you will receive ZERO (unless you have a valid reason, in which case you will need to appeal).**&#x20;

**EVEN IF THE TASK IS NOT FINISHED - SUBMIT ANYWAY! LOW MARKS > NO MARKS!**
{% endstep %}

{% step %}
### CELEBRATE!

This was a **HUGE** first project! Celebrate your tenacity and your resilience in sticking to it so early in the course!
{% endstep %}
{% endstepper %}
