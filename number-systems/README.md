# Number Systems

{% hint style="info" %}
## Learning Intention

* Investigate the use of number systems for computing purposes, including binary, decimal and hexadecimal
* Represent integers using two’s complement
{% endhint %}

{% hint style="success" %}
## Success Criteria

* **I can** compare and convert binary, decimal, and hexadecimal number systems.
* **I can** explain and apply two’s complement representation for integers.
{% endhint %}
